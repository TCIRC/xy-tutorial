HTML

HTML (hypertext markup language) 是構成基本網頁的元素

tag

一個 HTML tag 如下

<h1>Hello World</h1>

這是一個 h1 tag, 代表大標題
Hello World 為這個 tag 的內容

於瀏覽器上的效果如下

然後右鍵 inspect (檢查)


會看到

紅色的部分如下

<html>
    <head></head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>

可以看到瀏覽器幫我們補上了不少東西。

<body></body>中所放的是網頁的內容，所以我們的<h1></h1>被移入其中

也有很多其他不同的 tag 分別代表不同的意思
如<p></p> 代表 paragraph （段落）
<h2></h2>是小一點的標題

自動修正

瀏覽器自動修正錯誤的語法
若以下方

<html>
    <head></head>
    <body>
        <h2>Hello 次標題</H2>
        <p>正常的段落</p>
        <p>如果不
        <h5><p>最小的標題加上交錯tag</h5></p>
    </body>
</html>

會修正成

<html>
    <head></head>
    <body>
        <h2>Hello 次標題</h2>
        <p>正常的段落</p>
        <p>如果不</p>
        <h5><p>最小的標題加上交錯tag</p></h5><p></p>
    </body>
</html>

單獨的<p>補上了tag的結束（</p>)
交錯的tag：將<h5></h5>內部的<p>先結束，</h5>後面會只剩</p>，所以於前補上<p>，成為完整的tag
